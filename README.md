SauceDemo Automation Framework

github.com/Bettymali/SauceDemo\_Automation



A test automation framework built using Java and Selenium WebDriver 

to validate core user workflows. Designed for scalability, 

maintainability, and CI/CD integration.



\- Built Page Object Model framework with Maven, TestNG, and 

&#x20; Selenium WebDriver

\- Integrated Jenkins CI/CD pipeline running in Dockerized 

&#x20; environment with headless Chrome

\- Automated 3 login test scenarios with full pass/fail reporting

\- Version controlled with Git using feature branch and PR workflow
